Command line tool for running vlSLIC. Usage:

$ ../bin/vlslic_cli –help Allowed options: -h [ –help ] produce help message -i [ –input ] arg the folder to process (can also be passed as positional argument) -s [ –superpixels ] arg (=400) number of superpixels -m [ –minimum-region-size ] arg (=1) minimum region size allowed -c [ –compactness ] arg (=40) compactness = regularization trades off color for spatial closeness -t [ –iterations ] arg (=10) iterations -o [ –csv ] arg specify the output directory (default is ./output) -v [ –vis ] arg visualize contours -x [ –prefix ] arg output file prefix -w [ –wordy ] verbose/wordy/debug

Also see README.md for further documentation and default parameters.
